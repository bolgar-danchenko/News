# News

iOS app that uses NewsAPI as the source of news. 

### Features:
* News list
* Detail view
* Showing original article using WebKit
* Pagination (20 articles at once)
* Pull-to-refresh
* Network status check
* Caching fetched news
* Views count
* Dark mode

### Stack
* UIKit
* REST API
* JSON, Codable
* URLRequest Cache Policy
* WebKit
* UserDefaults
* DispatchQueue
